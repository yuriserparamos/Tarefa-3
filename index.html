Aspectos éticos e Legais da comunicação em sites.

Até mesmo no mundo digital, existem regras éticas e morais. Sejam elas respectivas as diretrizes de um site, ao manuzeio de dados, ou até mesmo a interação entre usuários. A ética no mundo digital atua para manter dignidade, segurança, privacidade dentre outros valores no ambiente virtual, seguindo tanto os valores morais quanto as legislações a respeito do assunto.

Função e utilidade gestão de projetos.

O gerenciamento de projetos é uma área que vem crescendo e tem sido cada vez mais introduzida dentre empresas de todos os tamanhos na atualidade. A gestão de projetos é importante, pois cada projeto é diferente, mas todos compartilham suas origens em uma ideia do que ser feito, e um potencial para um bom resultado se essa ideia é guiada em realidade.A gestão de projetos fornce uma estrutura para ajudar a alcançar os objetivos.

GitHub e Git

GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Ele permite que programadores, utilitários ou qualquer usuário cadastrado na plataforma contribuam em projetos privados e/ou Open Source de qualquer lugar do mundo. Você pode acessá-la ao clicar no ícone abaixo.
Git é um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo. Você será redirecinado ao site para realizar o download do git ao clicar no ícone abaixo.
